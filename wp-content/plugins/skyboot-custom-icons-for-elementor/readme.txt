== Description ==
Skyboot custom icons for elementor -  is a elementor based plugin. More than 14055+ Icons include in the plugin.
15 icon font pack are included in the plugin. you can use all icons on your websites. Also you can manage the icon pack separately. you can get ON/OFF button for individual icon pack.
Click for [Live Demo](https://skybootstrap.com/custom-icons-for-elementor)

If you are an Icon author feel free to contact me help.ukthemes [at] gmail.com. I will add your icon pack in this plugin.

= Plugin Features =

*	Fast loading.
*	15 icon font pack included.
*	ON/OFF button for every icon.
*	Easy to use.
*	Use icons anywhere

Please note: this plugin requires any version of the Elementor Page Builder to be installed and active. [Elementor is a free plugin](https://wordpress.org/plugins/elementor).


== Manual Installation ==

1.	Download the plugin
2.  Upload your plugin folder to the '/wp-content/plugins' directory.
3.	Unzip the plugin.
4.  Activate the plugin through the 'Plugins' menu in WordPress Dashboard.
